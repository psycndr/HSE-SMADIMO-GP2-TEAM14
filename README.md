# HSE-SMADIMO-GP2-TEAM14(32)




## Описание бизнес-задачи

Перед командой была поставлена задача подготовить базу для создания инструмента-помощника открытия ресторана в городе Нью-Йорк. Система сможет определить концепцию ресторана в Нью-Йорке, которая обеспечивает устойчивый потребительский спрос при минимальной конкурентной перегруженности сегмента - и тем самым минимизирует вероятность закрытия из-за низкой посещаемости в первые период после начала работы. 
Рынок ресторанов Нью-Йорка высококонкурентен и сильно сегментирован. Запуск без понимания структуры спроса, насыщенности сегментов и ожиданий целевой аудитории создаёт высокий риск провала даже при качественном продукте. Необходимо выбрать концепцию (кухня, ценовой сегмент, район), которая соответствует неудовлетворенному спросу рынка. Около 60% ресторанов закрываются в первый год - преимущественно из-за ошибочного выбора концепции или позиционирования, а не из-за операционных проблем.

Для создания основы такой платформы необходимо получить данные о уже открытых ресторанах, их предлагаемом ассортименте и рейтинге. Для сбора данных будут использоваться два самых популярных сервиса для оценок TripAdvisor и Yelp, они позволяют построить структуру спроса и конкуренции для принятия решений по формату и ценовой политике. 

## Описание хода выполнения проекта

В первую очередь был произведён сбор данных в датасет с двух платформ для оценки ресторанов и кафе.
Для сервиса Yelp использовался доступ через API (с использованием API ключа)
Для сервиса TripAdvisor данные собирались через скрейпинг, с эмуляцией действий реального пользователя, через библиотеку Selenium.
Далее была проведена очистка данных и проведён EDA, были выявленные основные закономерности и корреляции.

## Описание данных

Через API были получены 487 строк данных с id, названием, категориями (типы еды и ценовая) и рейтингом
Через парсинг были получены 10499 строк с рейтингом и категориями (типы еды и ценовая)

В ходе EDA была проведена работа с пропусками и дубликатами, датасеты были приведены в интерпретируемый вид, были построены графики зависимостей.
